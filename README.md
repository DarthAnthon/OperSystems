# Общее описание

Проект содержит библиотеку libcaesar, содержащую функции для установки ключа и шифрования/дешифрования, тестовую программу testpr, Makefile для сборки проекта и файл input.txt с входными данными.

# Описание функций

## Библиотека libcaesar:

void set_key(char key) - устанавливает ключ

void caesar(void* src, void* dst, int len) - производит шифрование/дешифрования содержимого буфера src и записывает результат в буфер dsr

## Программа test_pr:

int main(int argc, char* argv[]) - основная функция программы, производит динамическую загрузку библиотеки, получание адресов функций, чтение входного файла, установку ключа, шифрование и запись результата в файл.

# Описание команд

make - сборка проекта

make test - запуск теста
